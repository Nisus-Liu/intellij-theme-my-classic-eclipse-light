## 打包 & 部署

```
1. Build >> Build Project
2. cd .\out\production\intellij-theme-my-classic-eclipse-light\
3. jar -cf intellij-theme-my-classic-eclipse-light.jar ./
4. 从磁盘安装刚打包的主题插件
```
